# PageObjectFinal

## Requirements
- Python 3.10.10
- pytest==8.1.1
- selenium==4.18.1

A browser driver is required in the PATH directory. Chrome or Edge browsers are supported.

## Instalation

```bash
pip install -r requirements.txt
```
## Run tests

```bash
pytest -v -s --tb=line --language=en -m need_review test_product_page.py
```
